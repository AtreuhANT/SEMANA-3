# SEMANA-3

Responde a las preguntas por qué el método de Lagrange, Newton y matricial no responden bien. ¿Por qué el método de splines cúbicos lo hace bien?

Es pq el spline cúbico proporciona una mejor aproximación porque utiliza polinomios de bajo grado en cada intervalo y garantiza continuidad en la función, su primera y segunda derivada, evitando las oscilaciones que aparecen en los polinomios de alto grado como newton o llagrange.
